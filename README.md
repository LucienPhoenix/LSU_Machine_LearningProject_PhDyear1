# README

## CHE 7700 Project: Tennessee Eastman Process Fault Analysis for Chemical Process Automation - PhD Research

This project investigates process monitoring, fault detection, pattern discovery, and classification using data from the Tennessee Eastman Process (TEP), interpreted in the industrial context of the automated production of **acetic acid via oxidation of acetaldehyde**. The Tennessee Eastman Process is widely used as a benchmark in process systems engineering because it captures the complexity of real industrial operations, including multiple sensors, interacting units, nonlinear behavior, recycle effects, and dynamic fault propagation. In this study, the dataset is used as a process-automation proxy for understanding how modern machine learning can support industrial chemical manufacturing through better monitoring, diagnosis, and decision support.

The project uses **TEP00 as the normal operating benchmark** and treats **TEP01–TEP10 as faulty operating conditions**. This labeling strategy creates a clear process-monitoring framework in which the normal case represents healthy plant behavior and the remaining cases represent abnormal or faulted states. The goal is not only to distinguish normal from faulty behavior, but also to understand how different faults express themselves across the sensor network, how faults cluster in reduced-dimensional space, and how supervised models can classify operating conditions from measured process data.

A major focus of the work is **exploratory data analysis (EDA)**. Before building machine learning models, the project examines the structure of the TEP datasets, the number of sensors, the sample size per file, the dimensionality of the feature space, and the relationships among measured variables. This stage includes sensor count and dimensionality checks, summary statistics, variance analysis, correlation analysis, and fault-pattern visualization. These analyses help reveal which variables are stable, which vary strongly across operating states, and which sensors may be most informative for identifying abnormal plant behavior. Sensitivity-style analysis is also used to highlight variables that change most strongly between the normal benchmark and the faulted datasets.

The unsupervised learning component is designed to support **dimensionality reduction, visualization, and clustering-based interpretation**. The project considers several dimensionality-reduction methods, including **PCA, t-SNE, UMAP, PaCMAP, Dynamic PCA (DPCA), TriMAP, Multiple PCA (MPCA), and Autoencoders**. These methods are used to transform the high-dimensional sensor space into lower-dimensional representations that are easier to visualize and analyze. Once these reduced representations are obtained, clustering methods are applied to explore structure in the data. The main clustering approaches planned for this project are **K-means clustering, HDBSCAN, and Hierarchical clustering**. These combinations help reveal whether normal and faulty conditions separate cleanly, whether some fault cases overlap, and whether different methods produce consistent or contrasting views of the process states. The project also includes **Self-Organizing Maps (SOMs)** as an additional unsupervised tool for topology-preserving visualization of plant behavior.

The supervised learning component is intended to provide predictive fault-detection and fault-classification capability. The models emphasized in this project are **Random Forest, Gradient Boosting, and LSTM**. Random Forest and Gradient Boosting are included because they are strong nonlinear tabular-data models and can often provide variable importance information that supports industrial interpretation. The **LSTM** model is especially important because the Tennessee Eastman data are dynamic, and LSTMs are well suited to capturing time-dependent patterns in process measurements. This is particularly relevant in chemical reactors and recycle processes, where abnormal conditions may develop progressively rather than appearing as isolated static deviations. By comparing these supervised methods, the project aims to understand both classification performance and the extent to which temporal process behavior improves fault recognition.

From an engineering perspective, the project is framed as a contribution to **chemical process automation and intelligent monitoring**. In the industrial production of acetic acid via oxidation of acetaldehyde, maintaining stable operation, detecting abnormal behavior early, and understanding the source of process deviations are essential for safety, product quality, energy efficiency, and profitability. Although the Tennessee Eastman Process is a benchmark rather than a literal acetic acid plant model, it provides a realistic multivariable environment in which machine learning techniques can be evaluated for tasks that closely resemble industrial fault monitoring and diagnosis.

Overall, this project brings together **process understanding, data analysis, unsupervised learning, supervised learning, and engineering interpretation**. It is designed not only to build predictive models, but also to generate meaningful visualizations and explanations that connect machine learning outputs to chemical engineering decision-making. The final deliverables include processed datasets, analysis notebooks, figures, clustering and visualization results, model outputs, and written interpretation of the findings. The broader purpose is to demonstrate how machine learning can support modern process systems engineering by helping engineers detect faults, interpret plant behavior, and move toward more intelligent and resilient industrial automation.
